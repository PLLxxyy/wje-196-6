# 宠物合成养成 (Pet Merge & Collection Game)

一个基于 React 18 + TypeScript 的宠物合成收集游戏。

## 游戏特性

- **4x4 宠物格子**：拖拽相同等级宠物合成升级
- **10 个等级**：从蚂蚁到神龙，每个等级不同动物形象和稀有度
- **合成特效**：合成时有动画效果和音效反馈
- **宠物图鉴**：展示所有已收集和未解锁的宠物
- **每日签到**：每日签到送宠物蛋
- **金币系统**：通过出售宠物和完成成就获得金币，用于购买新蛋
- **成就系统**：稀有宠物收集达成特定成就，领取金币奖励
- **数据持久化**：所有数据保存在 localStorage

## 技术栈

- Vite 6
- React 18
- TypeScript
- localStorage 数据持久化
- 纯 CSS 样式（无第三方 UI 库）
